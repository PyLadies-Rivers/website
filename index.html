01 · ABOUT THE CHAPTER

A home base for Python in Rivers State

PyLadies is a global mentorship group with a mission to increase the number of women in tech through Python. Rivers Chapter is where that mission gets local — real classes, real projects, real people.

03 · CURRICULUM

What the 8 weeks cover

From first line of code to a finished project — the program is built as five progressive modules stretched across eight weeks of live sessions and practice.

WEEK 1–2

Basic Syntax

Variables, data types, operators, and writing your first working Python scripts.

WEEK 3

Data Structures

Lists, tuples, dictionaries, and sets — how to store and organize real information.

WEEK 4–5

Control Flow

Conditionals, loops, and functions, with practice exercises after every session.

WEEK 6

File Management

Reading and writing files, working with paths, and handling real data on disk.

WEEK 7–8

Build a Real Project

Everything comes together in a final, working project you build and present — plus mentor support to debug, polish, and finish strong.

Delivered through weekly live sessions, WhatsApp-based support, and the PyLadies Handbook — with Ada, our AI mentor, available for questions between sessions.
