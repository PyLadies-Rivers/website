# website
Official website for PyLadies Rivers — empowering women and underrepresented groups in Rivers State, Nigeria through Python education, mentorship, community events, and open source collaboration.

Update:  PyLadies Rivers July/August 2026 Python Training Has Begun

We are excited to announce that the PyLadies Rivers July/August 2026 Python Training Cohort officially started on 4 July 2026 and will run for 8 weeks.

This beginner-friendly program provides mentorship, weekly live sessions, practical assignments, and hands-on projects to help women build a strong foundation in Python programming.

We are grateful to the global PyLadies community for its continued inspiration and support, and we look forward to sharing updates and success stories from this cohort.
